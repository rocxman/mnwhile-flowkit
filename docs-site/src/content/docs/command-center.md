The Command Center is the keyboard-first launcher for the editor. It opens as a floating panel and is the fastest way to move between templates, search, imports, assets, layout, page management, layer management, and Studio-adjacent actions.

> [!NOTE]
> Open it with `Cmd + K` on macOS or `Ctrl + K` on Windows/Linux.

## Root commands

The top-level command list includes actions such as:

- Open Flowpilot
- Import from data
- Start from Template
- Edit Flow DSL
- Edit Mermaid Code
- Assets
- Search Nodes
- Auto Layout
- Design Systems

Playback-related entries may also appear when the relevant playback surface is active in the current product path.

## Views inside the Command Center

### Assets

Use Assets when you want reusable visual libraries, not generic editor actions. The current asset libraries are centered on:

- developer logos
- AWS service shapes
- Azure diagrams
- GCP diagrams
- CNCF shapes
- general-purpose icon sets

General insert actions are better handled from the toolbar add menu.

### Templates

Use templates when you want a proven starting structure instead of a blank canvas.

### Search

Use search to jump to named nodes in larger diagrams.

### Import

Use Import when you already have source material such as SQL, OpenAPI, Terraform, Mermaid, or code-oriented structure that should become an editable first draft.

### Layers

Use Layers when the diagram needs visibility, lock, ordering, or section-level organization without flattening everything into one canvas plane.

### Pages

Use Pages when one flow should contain multiple related canvases instead of separate documents.

### Design Systems

Use Design Systems to switch themes, duplicate them, edit them, or import/export theme JSON.

## How it fits with the rest of the editor

The Command Center is the fastest bridge between the canvas and Studio:

- use the canvas for direct manipulation
- use the inspector for exact edits
- use the Command Center for navigation, search, import, templates, assets, layers, pages, layout, and workflow switching
